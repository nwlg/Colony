PÁGINA DE INICIO:

La página de inicio será el índice que nos llevará al resto de las páginas. Es decir, desde esta página deberemos poder acceder al principio de cada una de las páginas que se crearán a continuación. También habrá que asegurarse de poder volver a la página de inicio desde cualquiera de las páginas creadas y por supuesto desde cualquier página a la anterior:
  • Página formateada con tablas para que simule la página de un periódico. hoja1
  • Página con más de una imagen en tamaño pequeño y texto explicativo, de forma que cada imagen esté enlazada para enviarme a la misma imagen en tamaño grande. hoja2
  • Página de formato de imagen donde se presentará la misma imagen en diferentes formatos gráficos, comentando las diferencias de calidad de imagen y tamaño de archivo que se aprecien. hoja3
  • Página donde se aprecien los efectos de giro y volteo de imágenes. hoja4
  • Página con la misma imagen en formato JPEG... hoja5
  • Página con una imagen y la misma imagen con alguno de sus colores transparentes. hoja6
  • Página donde se aprecien los efectos artísticos que se pueden aplicar con el irfanview. hoja7
  • Página con la misma imagen con diferente número de colores. hoja8
